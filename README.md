# Unemployment Analysis in India using Python

This project analyzes unemployment trends in India using Python.
The main focus is to understand how unemployment changed over time
and how COVID-19 affected employment during the lockdown period.

## Dataset
- Unemployment in India dataset
- Time period: May 2019 to June 2020
- Frequency: Monthly

## What was done
- Cleaned the dataset by removing missing values
- Converted date column for time-based analysis
- Visualized unemployment trends
- Compared unemployment before, during, and after COVID-19
